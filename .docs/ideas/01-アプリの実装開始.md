# 01-アプリの実装開始.md

作成日時: 2025-12-20:18:39

## 概要

- 必用な機能を満たすコマンドタイプのランチャーアプリがないため自作する

## 仕様

下記しようを満たすアプリを実装したい

- バックグラウンドに常駐してホットキーで呼び出し
- フルスクリーンモードでのホットキーの無効化
- ホットキー無効化時にタスクトレイのアイコンを変更
- 実行コマンドはすべて設定ファイル(`~/.config/<APP_NAME>/setting.yaml`)で定義
- 環境変数ファイル(`~/.config/<APP_NAME>/env.yaml`)も読み込み設定ファイルの実行コマンドに対してパスを渡す

## 技術選定

- Win32 API 操作：windows (windows-rs) crate
- グローバルホットキー： crate `global-hotkey`（要評価）
- タスクトレイ：`tray-icon` / `tray-item` / Windows API（NOTIFYICONDATA）
- UI：`egui`（eframe）または `slint`
- TOML 設定：`toml` / `serde`
- 実行：`std::process::Command`
